UFO Sightings Analytics

Sobre o Projeto

O UFO Sightings Analytics é uma aplicação de análise de dados desenvolvida em Python. 
O objetivo principal deste projeto é processar e visualizar registros históricos de avistamentos de OVNIs, 
transformando dados brutos em insights visuais através de uma interface web interativa. Este projeto reflete 
meu aprendizado contínuo na análise de dados e na estruturação de aplicações modulares.

Tecnologias e Ferramentas

O projeto utiliza um ecossistema focado em ciência de dados e desenvolvimento ágil:

* Interface Web: Construída com Streamlit, permitindo uma navegação intuitiva pelos resultados.
  
* Manipulação de Dados: Pandas e NumPy foram fundamentais para a limpeza, tratamento e estruturação do dataset original.
  
* Visualização: Utilização de Matplotlib e Seaborn para gerar gráficos que facilitam a interpretação dos avistamentos.

* Arquitetura: O projeto foi organizado em módulos (ui/, data/, config/), demonstrando uma preocupação com boas práticas de desenvolvimento de software e legibilidade do código.
  
Funcionalidades Principais

* Pipeline de Dados: Processamento automatizado de arquivos CSV para extração de tendências.

* Dashboard Interativo: Interface que permite filtrar e explorar diferentes dimensões dos avistamentos de forma dinâmica.

* Modularidade: Estrutura de código que facilita a manutenção e a escalabilidade.
